Resources for the AutoSplit (https://github.com/Toufool/AutoSplit) and LiveSplit (https://github.com/LiveSplit/LiveSplit) tools specifically for the game Nubby's Number Factory by MogDogBlog.
You will have to specify the path to your AutoSplit integration files on LiveSplit and the Image and Screenshot folders for AutoSplit.

AutoSplit Resources contains a settings file that uses the round number window on the right of the screen to track and split your run.

Each splitting option will contain a folder with the images that AutoSplit uses, and a Split and Layout file for LiveSplit.

Split by Zone - Splits the game by the 4 zones (20 round blocks). Resets upon seeing the pause menu.

Split by Zone Challenges - Same as above but uses a mildly different image for run end as challenges displays differently due to not having endless mode.
